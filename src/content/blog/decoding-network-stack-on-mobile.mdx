# Introduction

I have been working on creating a web filtering feature for the Adora Kids App in my internship and this led me on one of the biggest rabbit holes I've ever encountered. In this blog I'll share my research on how to navigate around the networking protocols on iOS (`PacketTunnelProvider`) and Android (`VpnService`).

Both Android and iOS implement a layered IP stack (L3/L4) above radio/link layers. Apps normally hand off sockets to the OS, which resolves DNS, opens TCP/UDP flows, and routes packets via system interfaces. A “VPN” on mobile isn’t just encryption, it’s an interception point: the OS creates a virtual network interface (a TUN device) and routes selected traffic into your process. From there you can read raw IP packets, make policy decisions (block, forward, modify), and write packets back to the interface.
