# Hyena Dart
Hyena Dart is a powerful Flutter/Dart codebase analyzer that detects dead code and calculates code complexity metrics. Built using the official Dart analyzer package for accurate AST-based analysis.

## v1.0 Milestone

v1.0 is the first release of Hyena Dart.
